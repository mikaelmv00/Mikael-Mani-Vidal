Project 1. | Creating this website

Objective

The objective of this project was to create a interactive website that contained the worksheet given in class, VÉL608G, using HTML5 to then write in detail how the website was created. The website should comprise of my contact information as well as my resume. Then the website should be published on Github using Git bash. The website was allowed to be written in Icelandic or English. It was important to address the problems and obstacles that I encountered during the development of this website and how I resolved those issues.

Preparation

I started off by learning the basics from lectures that were provided by my professor. Then, I researched and found information about how to write HTML code at W3Schools.com. After that, I downloaded the program Brackets, along with Git-SCM, and created an account on github. After I signed up, I clicked New in the repositories column on the left side of the homepage.

I filled in the required fields Repository name with a name that I wanted to use for the website and made sure that i chose the Public option, not Private

I created a logo for the website that I wanted on the front page and as a favicon.

Creating the website

I found a free open source template called "Dimension" on html5up. I downloaded it and then uploaded the template to Brackets by dragging the Index file over.

Using my newly developed HTML coding skills, I edited the HTML code from the template in Brackets. I utilized the "Live Preview" feature to see the changes I made. Whenever I modified anything in the code, I simply pressed CTRL + S to save the index file, and the website updated automatically.

Project 2 | Computer-aided incision

Objective

The objective of this project was:

  1. To use a vinyl cutter to create a sticker.
  2. To design a parametric, press fit model of building blocks. The model should be scalable.

Methodology

Sticker

For the sticker, I searched for sticker ideas and designs online. I came across one that I found funny, so I chose that design.

To cut it out using the vinyl cutter, the image needed to be saved in .svg format. I used the website Convertio to convert it.

Next, I downloaded the program Inkscape, uploaded the design file, connected the computer to the printer, and printed the sticker.

Here are the results:

Press fit model

Methodology

For this part of the project, I was assigned to design a model consisting of pieces for manual press-fit assembly. Taking advantage of the opportunity, I decided to create a new ceiling light for my living room.

I started by searching the internet for inspiration, using keywords such as 'modern light designs,' and focused on the ones that were simplistic and elegant. After sketching some ideas, I opened Inventor to begin the design process.

Since I already had the existing infrastructure for my old ceiling light, I took measurements and designed the light shade to fit around the original foundation.

I started off by sketching the measurements of the structure in Inventor, and designed the central light disk to hold the light.

I set the parameters and worked on the central light disc. I set them by going to Manage -> Parameters. There, I wrote down all the parameters for my parts. By defining the parametes for the part, it makes it easier to change various lengths as well gives me the ability to scale up or down the part.

Here is how the disc turned out.

Then, in order to create a disc allowing for self-assembly I cut out 20 small rectangles along the border of the disc, all at a 40 degree incline at 18 degree increments using the 'pattern' feature in Inventor. It was necessary to have 40 degree inclnes so that the attachable 'wings' or the outer shade could provide some design and of course provide shade and control the amount of light that comes through.

The end results looked like this:

Then I designed a "wing" that connected to the disc and holds the light together.

Then I made another disc that was supposed to be at the bottom, which looked similar to the first one. Then I created an assembly and the design was complete.

Finally, the only thing left to do was export the sketches as a .DXF file and upload to Inkscape, which then allows for the designs to be laser-cut from provided material. Of course, as any good engineer can attest to, miscalculations and cost inefficient designs run rampant, and it was discovered the original light design would require 12 sheets of expensive material resulting in 30.000 krona worth of expenses. I then decided to redesign the light such that both material and cost would decrease, resulting in a similar design but with a simpler shape.

Below is the first more cost-efficient prototype.

That looked like this in Inkscape.

Then I made about 8 more versions where I maximized the size of the wing and the efficiency of the material use, while still maintaining its' elegance. I could have originally made the wings entirely rectangular for optimal cost efficiency, but to keep the design a more aesthetic looking piece, I retained the curvature of the wing.

Final words

I was not as happy with this design as the original one, but at least it was really cheap to produce. Now I needed only three of 50x50 plywood panels instead of 12. The project went really smooth and I thought that this was a very fun and interesting and i learned a lot from it. The design design documentations can be downloaded here below.

Project 3. | 3D Scanning and 3D printing

Be aware, this site is still under construction and this project is not finished!

Objective

The objective of this project was to learn how to 3D scan an object and how to use a 3D printer.

Methodology

3D printing

Now that I have installed a new ceiling light in my living room, I wanted to design and 3D print a tea light holder. I needed some decoration for my home, so I decided to 3D print a self-glowing tealight holder. The idea was to maximize the energy usage from the tealight by capturing the energy from the flame and storing it in the holder. Once the tealight runs out, we can still have a light source from the candle holder. I wanted a tea light holder that resembled a part of a machine to give it an industrial vibe. While searching online, I came across a part for a KitchenAid mixer and another part that is used as a screw cap, which would be perfect for my tea light holder.

I opened Inventor and immediately started designing. Firstly, I looked up the diameter of a tealight and and proceeded to draw three circles. The first circle had a slightly larger diameter than the candle, the second circle had a diameter 5 mm wider than the first one, and the third circle had a diameter 10 mm wider than the first one. I extruded these circles to give them depth and create the desired shape. Next, to resemble the round cap I found online, I added threads and incorporated a few holes into the object to allow better airflow for the candle. Here is the final result of my design.

I thought that this light holder looked awasome and that it would fit well in my apartment. Now, I needed to save the file in .STL format. I did that by following these steps: I clicked on "Save As," then selected "Save Copy As." In the "Save as type" dropdown menu, I chose "STL Files (*.stl)".

Next, I needed the PrusaSlicer program, so I installed it on my computer. Once installed, I opened the program and imported my .STL file by selecting the "Add" option.

Then, I proceeded to select my printer settings and utilized the following settings:

Now, I exported my files to a memory SD card, Inserted the card to the printer and printed the object. The material that I used for my tealight holder was green glow in the dark plastic. This is how it turned out.

As evident from the results, the printing process encountered a failure as the printer did not successfully print out the threads. Additionally, a design flaw became apparent after using the candlelight holder: the candle burned at a higher temperature than the plastic, resulting in the gradual softening and potential melting of the material. To address this issue, an alternative solution would be to utilize LED candles, thereby eliminating the problem entirely. Alternatively, to prevent such issues in the future, I could have positioned the candle higher up in the holder to allow for better heat dissipation into the surrounding air, reducing the impact on the tea light holder.

Despite these setbacks, I am satisfied with the overall outcome of the project. The candle holder possesses an appealing aesthetic when illuminated in the darkness.
