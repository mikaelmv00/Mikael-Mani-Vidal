Portfolio

Project 1. | Creating this website

Objective

The objective of this project was to create a interactive website that contained the worksheet given in class, VÉL608G, using HTML5 to then write in detail how the website was created. The website should comprise of my contact information as well as my resume.

Methodology

Step 1.

I learned the basics from lectures that were provided by my professor.

Step 2.

I researched and found information about how to write HTML code at W3Schools.com.

Step 3.

I downloaded the program Brackets, along with Git-SCM, and github.

Step 4.

I found a free open source template at html5up. I uploaded the template to Brackets and used my newly developed skill in HTML coding to edit the template for my needs.

Step 5.

I edited the HTML code from the template in the program Brackets and used the live "Live Preview" feature to see the changes I made.

Step 6.

I created a logo for the website.

Step 7.

I made the logo as the browser tab icon (favicon) for the website by writing: < link rel="icon" href="images/logo5.png" > , where "images/logo5.ping" is the path to the logo, in the head of the website.

Step 8.

I created a GitHub account at github.com and created a new repository, uploaded all my files to there by pressing "upload" and dragging them over. I ended up not having to use Git-SCM to upload my files to GitHub, which made the whole process easy and straight forward.

Step 9.

I uploaded the website by going to settings, and opening "pages" under "Code and automation". There I pressed the drop-down list under "Branch" and changed it from "None" to "main"

Finally, my website was completed! Not the only thing to do is to copy the link to my website and share it!

Project 2. | Computer-aided incision

Objective

The objective of this project was:

1. To use a vinyl cutter to create a sticker.

2. To design a parametric, press fit model of a building blocks. The model should be scalable.

Methodology

Sticker

For the sticker, I Googled sticker ideas and searched for designs. Then, I found one that I thought was funny, so I picked that one.

In order to cut it out with the vinyl cutter the image had to be saved in .svg format. So I used the website Convertio for that. Then, I downloaded the program, Inkscape, uploaded the design file and connected the computer to the printer then printed the sticker.

These are the results:

Press fit model

Methodology

Here, I was tasked with designing a model that consisted of peices for later manual pressfit-assembly. Since I needed a new ceiling light in my living room, I decided to strike two birds with one stone and make that as my project.

I began by finding inspiration through internet search, using keywords such as 'modern light designs' and honed in on the ones I found most simplistic and elegant. Then I sketched down some ideas and opened up Inventor.

I still had my old ceiling light infrastructure, so I measured it and designed my light shade around the original foundation.

I started off by sketching the measurements of the structure in Inventor, and designed the central light disk to hold the light.

I set the parameters and worked on the central light disc.

Here is how the disc turned out.

Then, in order to create a disc allowing for self-assembly I cut out 20 small rectangles along the border of the disc, all at a 40 degree incline at 18 degree increments using the 'pattern' feature in Inventor. It was necessary to have 40 degree inclnes so that the attachable 'wings' or the outer shade could provide some design and of course provide shade and control the amount of light that comes through.

The end results looked like this:

Then I designed a "wing" that connected to the disc and holds the light together.

Then I made another disc that was supposed to be at the bottom, which looked similar to the first one. Then I created an assembly and the design was complete.

Finally, the only thing left to do was export the sketches as a .DXF file and upload to Inkscape, which then allows for the designs to be laser-cut from provided material. Of course, as any good engineer can attest to, miscalculations and cost inefficient designs run rampant, and it was discovered the original light design would require 12 sheets of expensive material resulting in 30.000 krona worth of expenses. I then decided to redesign the light such that both material and cost would decrease, resulting in a similar design but with a simpler shape.

Below is the first more cost-efficient prototype.

That looked like this in Inkscape.

Then I made about 8 more versions where I maximized the size of the wing and the efficiency of the material use, while still maintaining its' elegance. I could have originally made the wings entirely rectangular for optimal cost efficiency, but to keep the design a more aesthetic looking piece, I retained the curvature of the wing.

I was not as happy with this design as the original one, but at least it was really cheap to produce. Now I needed only three of 50x50 plywood panels instead of 12.

Project 3. | 3D Scanning and 3D printing

Be aware, this site is still under construction and this project is not finished!

Objective

The objective for this project was to learn how to 3D scan an object and to learn how to use a 3D printer.

Methodology

3D printing

I had very lite time to finish this project, so I decided to design a very basic object to 3D print. I needed some decoration in my home, so I decided to 3D print a simple tealight holder.

I opened up Inventor and went straight to designing. First, I looked up the diameter of a tealight, and then I drew 3 circles. One that had a little bigger diameter than the candle, another one that had 5 mm wider diameter than the first circle, and the third one had 10 mm wider diameter than the first one. I extruded this to form the shape, then I wanted some texture on the surface, so I made it threaded and then I made few holes in the object. This is how it turned out.

This is not my best work, and as I said, I was very tight on time, so I spent only around 5 minutes to create this object.

Now, I needed to save the file of .STL form. I did that by pressing Save As --> Save Copy As --> Save as type --> STL Files(*.stl)

Now, I needed the program PrusaSlicer, so I installed it on my computer. There, I imported my .STL file into the program by selecting 'Add'.

Then, I selected my printer settings, and used these settings:

Now, I exported my files to a memory SD card, Inserted the card to the printer and printed the object. The material that I used for my tealight holder was green glow in the dark plastic.
