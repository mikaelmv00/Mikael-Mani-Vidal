Intro

This website is produced within the framework of VÉL608G, a course at the University of Iceland, taught by Hafliði Ásgeirsson, spring semester 2023. The aim of the site is to keep a record of all projects throughout the semester, as well as to serve my personal portfolio.

Portfolio

Project 1. | Creating this website

Objective

The objective of this project was to create a functional website that hosted the worksheet of the class, VÉL608G, using HTML5 and write in detail how the website was created. The website should also have my contact information as well as my resume.

Methodology

Step 1.

I learned the basics from lectures that was provited by my professor.

Step 2.

I researched and found information about how to write HTML code at W3Schools.com.

Step 3.

I downloaded the program brackets, along with git-scm, and github.

Step 4.

I found a free open source template at html5up. I uploaded the template to Brackets and used my newly developed skill in HTML coding to edit the template to my needs.

Step 5.

I created a logo for the website

Step 6.

I created a github account at github.com and created a new repository, uploaded all my files to there by pressing "upload" and dragging them over.

Step 7.

I uploaded the website by going to settings, and opening "pages" under "Code and automation". There I presset the drop down list under "Branch" and changed it from "None" to "main"

Finally, my website was completed! Not the only thing to do is to click the link to my website and share it!
